UBC Course Registration Worklist

The Worklist stores courses to be registered later. Through a Worklist, the student can compare multiple schedules in order to get the most convenient timetable.

The objective of this prototype is to improve the following features inside of the current worklist

  • Browse of multiple Worklists
  • Display of the Timetable elaboration
